Deep Learning Experiments
This repository contains my Deep Learning experiments completed as part of my academic coursework. The focus of this work is to build a strong practical understanding of how deep learning models are designed, trained, and evaluated.

## What this repo covers

* Implementation of basic **Neural Networks**
* Working with **Convolutional Neural Networks (CNNs)**
* Model training, validation, and performance evaluation
* Understanding overfitting, underfitting, and optimization techniques
* Experimenting with different architectures and hyperparameters

Objective

The goal of this repository is to move beyond theoretical concepts and gain hands-on experience with deep learning models by experimenting with real datasets and observing model behavior in different scenarios.

## Key Learnings

* How neural networks learn from data
* Importance of data preprocessing and normalization
* Effect of hyperparameters on model performance
* Practical challenges in training deep learning models

## 📌 Note
This repository reflects my learning journey in Deep Learning and is continuously evolving as I explore more advanced concepts and techniques.
